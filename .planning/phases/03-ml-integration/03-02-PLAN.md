<task type="auto">
  <name>Task 2: Congestion prediction model</name>
  <files>src/ml/congestion_model.py</files>
  <action>
    Create src/ml/congestion_model.py:

    - CongestionLevel enum: LOW, MODERATE, HIGH, CRITICAL

    - AreaCongestion dataclass:
      - area_id: str (e.g., "runway_28L", "taxiway_A", "terminal_A")
      - area_type: str (runway/taxiway/terminal/apron)
      - level: CongestionLevel
      - flight_count: int
      - predicted_wait_minutes: int
      - confidence: float

    - CongestionPredictor class:
      - __init__()
      - Define airport areas with capacities:
        - runway_28L: capacity 2 (landing/takeoff)
        - runway_28R: capacity 2
        - taxiway_A: capacity 5
        - taxiway_B: capacity 5
        - terminal_A_apron: capacity 10
        - terminal_B_apron: capacity 10

      - _count_flights_in_area(flights: List[dict], area: str) -> int:
        - Use flight position to determine area
        - Ground + low altitude near runway = runway area
        - Ground + heading toward terminal = taxiway
        - Ground + stationary = apron

      - predict(flights: List[dict]) -> List[AreaCongestion]:
        - Count flights per area
        - Compute congestion level based on capacity ratio
        - LOW: <50%, MODERATE: 50-75%, HIGH: 75-90%, CRITICAL: >90%
        - Estimate wait time based on level

      - get_bottlenecks(flights: List[dict]) -> List[AreaCongestion]:
        - Return only HIGH and CRITICAL areas

    - predict_congestion(flights: List[dict]) -> List[AreaCongestion]:
      - Convenience function using default predictor
