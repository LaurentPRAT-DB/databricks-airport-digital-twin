  <action>
    Create src/ml/delay_model.py:

    - DelayPrediction dataclass:
      - delay_minutes: float
      - confidence: float (0-1)
      - delay_category: str (on_time/slight/moderate/severe)

    - DelayPredictor class:
      - __init__(model_path: Optional[str] = None)
      - Uses simple rule-based logic for demo (no external ML dependencies needed)
      - Rules based on:
        - Time of day: peak hours (7-9am, 5-7pm) +10-20 min base delay
        - Altitude: ground aircraft more likely delayed
        - Historical patterns: random factor for realism

      - predict(features: FeatureSet) -> DelayPrediction:
        - Calculate base delay from rules
        - Add noise for realism (±5 minutes)
        - Compute confidence (higher for ground, lower for cruise)
        - Categorize: <5 min = on_time, <15 = slight, <30 = moderate, else severe

      - predict_batch(flights: List[dict]) -> List[DelayPrediction]:
        - Extract features and predict for each flight

    - predict_delay(flight: dict) -> DelayPrediction:
      - Convenience function using default predictor
