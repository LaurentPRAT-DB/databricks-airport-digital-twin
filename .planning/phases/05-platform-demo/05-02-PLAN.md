# Plan 05-02: Platform Integration (Lakeview, Genie, Lineage)

## Overview

**Goal**: Integrate Databricks platform features to showcase the full ecosystem.

**Requirements**: PLAT-01, PLAT-02, PLAT-03, PLAT-04 (✅ complete)

**Dependencies**: Phase 1 (Data), DABs deployment

## Success Criteria

1. Lakeview dashboard displays real-time flight metrics (embedded or linked)
2. Genie Space configured for natural language flight queries
3. Data lineage view accessible showing pipeline flow
4. All integrations work in deployed Databricks App

## Tasks

### Task 1: Lakeview Dashboard Creation

**Files to create:**
- `dashboards/flight_metrics.lvdash.json` - Dashboard definition
- `resources/dashboard.yml` - DAB resource for dashboard

**Implementation:**
1. Create Lakeview dashboard with flight metrics:
   - Total active flights (counter)
   - Flights by phase (bar chart: ground/climbing/cruising/descending)
   - Average delay by airline (bar chart)
   - Gate utilization (gauge)
   - Flight positions over time (time series)
2. Query data from Gold tables in Unity Catalog
3. Add dashboard to DAB resources for deployment
4. Add link/embed in frontend sidebar

### Task 2: Genie Space Setup

**Files to create:**
- `databricks/genie_space_config.json` - Genie space configuration

**Implementation:**
1. Create Genie Space pointing to flight data tables:
   - gold_flight_positions
   - gold_gate_status
   - gold_predictions
2. Configure sample questions:
   - "How many flights are currently in the air?"
   - "Which gates are available in Terminal A?"
   - "What flights have predicted delays over 15 minutes?"
   - "Show me all Alaska Airlines flights"
3. Add Genie link to frontend header or sidebar

### Task 3: Data Lineage Integration

**Files to modify:**
- `app/frontend/src/components/Header/Header.tsx` - Add lineage link

**Implementation:**
1. Generate Unity Catalog lineage URL for pipeline tables
2. Add "View Lineage" button in header that opens lineage graph
3. URL format: `{workspace}/explore/data/{catalog}/{schema}/{table}/lineage`

### Task 4: Platform Links Component

**Files to create:**
- `app/frontend/src/components/PlatformLinks/PlatformLinks.tsx`

**Implementation:**
1. Create sidebar or dropdown with platform links:
   - "📊 Flight Dashboard" → Lakeview dashboard
   - "🗣️ Ask Genie" → Genie Space
   - "🔗 Data Lineage" → Unity Catalog lineage
   - "📈 MLflow Experiments" → MLflow UI
2. Links open in new tabs
3. Environment-aware URLs (dev vs prod workspace)
